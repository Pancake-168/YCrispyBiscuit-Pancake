# 基础组件规范

## 前置：现有资产清单

以下内容已在 `index.css` 和 `variables.css` 中实现，组件直接使用，无需重复建设：

| 资产 | 文件 | 状态 |
|---|---|---|
| `.glass` | index.css | ✅ 毛玻璃面板基类 |
| `.btn.primary` | index.css | ✅ 渐变主按钮 |
| `.btn.secondary` | index.css | ✅ 点缀色边框按钮 |
| `.btn.subtle` | index.css | ✅ 透明低存在感按钮 |
| `.btn.danger` | index.css | ✅ 危险操作按钮 |
| `.btn.spinning` | index.css | ✅ 加载旋转态 |
| `.icon-btn` | index.css | ✅ 图标按钮 |
| `.pill.success / .error / .warn / .info` | index.css | ✅ 状态标签 |
| 设计 Token（`--bg`、`--text`、`--muted`、`--accent` 等） | variables.css | ✅ |

**组件 CSS 规则**：颜色、圆角、阴影、模糊值必须引用 `var(--xxx)`，禁止写死值。需要新 Token 时先在 variables.css 中定义，语法遵循项目命名规则（语义命名，不描述实现）。

**模糊值层次**：项目统一使用三级模糊，定义在 `:root`：
| Token | 值 | 用途 |
|---|---|---|
| `--blur-overlay` | `4px` | 遮罩层（Dialog 遮罩） |
| `--blur-panel` | `10px` | 面板/卡片（`.glass`、Popover、DropdownMenu、Select 面板、Toast） |
| `--blur-tooltip` | `12px` | 浮层提示（Tooltip） |

**已有全局类中 `.glass` 的 `backdrop-filter` 也需改为 `blur(var(--blur-panel))`**。

## §1 Button

封装 `loading` 态的按钮组件，避免每个调用方重复写 spinning + disabled + 文字替换逻辑。

```tsx
<Button variant="primary" loading={isSubmitting} onClick={handleSubmit}>
  提交
</Button>
```

| 传参 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `variant` | `"primary" \| "secondary" \| "subtle" \| "danger"` | `"primary"` | 对应全局 `.btn.*` 类 |
| `disabled` | `boolean` | `false` | |
| `loading` | `boolean` | `false` | 为 true 时自动追加 `.btn.spinning`、设置 `disabled`、用 `loadingText` 替换 `children` |
| `loadingText` | `string` | `"请稍候..."` | loading 时显示的文字 |
| `onClick` | `() => void` | — | |
| `icon` | `ReactNode` | — | 前置图标，放在文字左侧 |
| `children` | `ReactNode` | — | 按钮文字 |
| `type` | `"button" \| "submit"` | `"button"` | |
| `className` | `string` | — | 追加自定义类名 |

**实现方式**：内部渲染 `<button className={`btn ${variant}`}>`，`loading` 时叠加 `spinning` 类 + `disabled` 属性。组件只负责行为收敛，样式完全来自全局 CSS 类，不需要 module.css。

## §2 IconContainer

统一图片/图标容器。无论原图多大（包括 1024px 级别），容器保持固定显示尺寸，图片自适应居中裁剪。

| 传参 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `size` | `number` | — | 容器宽高（px），正方形 |
| `shape` | `"rounded" \| "circle"` | `"rounded"` | `rounded` = `var(--radius-md)` 圆角方形，`circle` = `50%` 圆形 |
| `src` | `string \| ReactNode` | — | 传字符串时渲染 `<img>`，传 React 图标组件时直接渲染该组件 |
| `alt` | `string` | `""` | `src` 为字符串时用作 `<img alt>`，为 ReactNode 时忽略 |
| `fallback` | `ReactNode` | 灰色占位图标 | `src` 为字符串且加载失败时显示 |
| `className` | `string` | — | |

**行为**：`src` 为字符串时使用 `<img>` + `object-fit: cover` 填满容器；为 ReactNode 时直接渲染，居中于容器。容器背景 `var(--glass)`，边框 `1px solid var(--glass-brd)`。module.css 仅处理容器尺寸和裁剪。

## §7 Tooltip

悬停提示。鼠标停留指定时间后展示文字说明。

| 传参 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `content` | `string` | — | 提示文字，仅文字不嵌套组件 |
| `children` | `ReactNode` | — | 被包裹的触发元素 |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"top"` | 弹出方向，贴近视口边缘时自动翻转 |
| `delayDuration` | `number` | `300` | 悬停延迟（ms） |

**Radix 底层**：`@radix-ui/react-tooltip`。

**样式**：`background: var(--tooltip-bg)` + `color: var(--tooltip-text)` + `backdrop-filter: blur(var(--blur-tooltip))` + `border: 1px solid var(--tooltip-border)` + `border-radius: var(--radius-sm)` + 小三角形箭头（fill 同 `var(--tooltip-bg)`）。Tooltip 不跟随主题切换，亮/暗色下保持同一套深色值。

**需新增 Token**（variables.css，不跟随 data-theme，定义在 `:root` 层）：`--tooltip-bg: rgba(32,32,32,0.92)`、`--tooltip-text: rgba(255,255,255,0.93)`、`--tooltip-border: rgba(255,255,255,0.12)`。

## §9 DropdownMenu

下拉菜单，展示操作项列表。

| 传参 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `trigger` | `ReactNode` | — | 触发元素 |
| `items` | `MenuItem[]` | — | 菜单项数组，见下方 |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | |
| `align` | `"start" \| "center" \| "end"` | `"start"` | |

```ts
interface MenuItem {
  label: string;
  icon?: ReactNode;       // 左侧图标
  shortcut?: string;      // 右侧快捷键文字（如 "⌘K"）
  disabled?: boolean;
  danger?: boolean;       // true 时文字为 error 色
  separator?: true;       // true 时该项渲染为分隔线，忽略 label/onClick
  onClick?: () => void;
}
```

**Radix 底层**：`@radix-ui/react-dropdown-menu`。

**行为**：点击触发打开，选中项后关闭，键盘 ↑↓ 导航、Enter 确认、Esc 关闭。

**样式**：菜单面板同 Popover 卡片（引用 `.glass` Token）。项布局 `[图标] 文字 [...快捷键]`。hover 项 `background: rgba(var(--accent-rgb), 0.08)`。danger 项文字 `var(--color-error)`。分隔线 `1px solid var(--glass-brd)`。module.css 仅处理面板定位和项内布局。

## §11 Dialog

通用弹窗。遮罩层模糊背景，焦点锁定在弹窗内，Esc 关闭。

| 传参 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `trigger` | `ReactNode` | — | 触发元素，不传则由外部 state 控制 |
| `open` | `boolean` | — | 外部控制开闭 |
| `onOpenChange` | `(open: boolean) => void` | — | |
| `title` | `string` | — | 弹窗标题 |
| `description` | `string` | — | 副标题，显示为小号 uppercase muted 文字 |
| `children` | `ReactNode` | — | 正文内容，可自由组合任意下层组件（Button/Input/Select/Tabs 等） |

**Radix 底层**：`@radix-ui/react-dialog`。

**样式**：遮罩层 `background: var(--dialog-overlay-bg)` + `backdrop-filter: blur(var(--blur-overlay))`。弹窗主体引用 `.glass` 对应的全套 Token + `box-shadow: var(--shadow-lg)` + 入场 elastic 缩放动画（`cubic-bezier(0.34, 1.56, 0.64, 1)`）。module.css 仅处理遮罩、主体居中定位和动画。

## §13 Toast

| 传参 | 类型 | 默认值 | 说明 |

| `duration` | `number` | `3000` | 自动消失时长（ms） |
| `action` | `{ label: string; onClick: () => void }` | — | 可选操作按钮（如撤销） |

**Radix 底层**：`@radix-ui/react-toast`（需在 App.tsx 根节点引入 `Toast.Provider`，包裹整个应用，使所有 Toast 调用方共享同一通知通道）。

**样式**：Toast 容器 `background: var(--bg-elev)` + `border: 1px solid var(--glass-brd)` + `backdrop-filter: blur(var(--blur-panel))` + `box-shadow: var(--shadow-md)` + `border-radius: var(--radius-md)`。左边色点直接用全局 `.pill.{variant}` 类（复用 pill 的色值逻辑）。入场动画：从右侧滑入 + 淡入。

## §15 ScrollArea

统一样式的滚动容器，替换原生滚动条。

| 传参 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `children` | `ReactNode` | — | |
| `maxHeight` | `number \| string` | — | 最大高度，超出可滚动 |
| `className` | `string` | — | |

**Radix 底层**：`@radix-ui/react-scroll-area`。

**部件结构**（Radix ScrollArea 固定嵌套模式，实现时必须配齐）：
```tsx
<ScrollArea.Root>
  <ScrollArea.Viewport>       {/* 可视区域 */}
    {children}                {/* 内容 */}
  </ScrollArea.Viewport>
  <ScrollArea.Scrollbar orientation="vertical">   {/* 垂直滚动条 */}
    <ScrollArea.Thumb />      {/* 滑块 */}
  </ScrollArea.Scrollbar>
</ScrollArea.Root>
```

**样式**：Viewport 设置 `maxHeight` 限制高度。Scrollbar 宽度约 8px，Thumb 背景 `var(--glass-brd)` + `border-radius: 4px` + hover 时 `var(--accent)`。样式与全局 `::-webkit-scrollbar` 保持一致。
